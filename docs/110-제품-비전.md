# 제품 비전

## 목적
- Lifebase의 제품 방향과 핵심 원칙을 정의한다.

## 제품 정의
- 클라우드 스토리지 + 캘린더 + 할 일을 하나로 통합한 개인 생활 관리 서비스
- Google 계정 기반 인증으로 진입 장벽을 낮춤

## 타깃 사용자
- 지인 대상 서비스
- 각 사용자는 독립된 공간을 가지며, 다른 사용자의 정보를 알 수 없어야 한다
- 사용자 간 정보 격리가 핵심 원칙

## 핵심 차별점
- 기획 중(추후 확정)

## 구현 방침
- MVP 단계를 두지 않고, 문서에 정의된 전체 기능을 처음부터 구현한다
- 플랫폼: Web, Desktop(macOS/Windows), Mobile(iOS/Android) 동시 지원

## 열린 질문
- 경쟁 대비 핵심 차별점 확정 시점
