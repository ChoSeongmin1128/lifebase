# UX/UI 테마

## 목적
- 버튼, 간격, 모서리, 폰트 등 UI 컴포넌트의 통일 규칙을 정의한다.

## 상태
- 추후 상세 기획 예정

## 버튼 체계 (초안)
- Primary: 브랜드 컬러 배경, 흰 텍스트 (주요 액션: 저장, 생성)
- Secondary: 회색 배경 또는 아웃라인 (보조 액션: 취소, 닫기)
- Danger: 빨간 계열 (삭제, 해제)
- Ghost: 배경 없음, 텍스트만 (덜 중요한 액션)
- 한 화면에 Primary 버튼은 1개만

## 간격/크기 (초안)
- 4px 또는 8px 배수 그리드
- 모서리: 8px (rounded-lg) 통일

## 아이콘
- 아이콘 + 텍스트 병행 (아이콘만으로 의미 전달 부족)

## 열린 질문
- 아이콘 라이브러리 선택 (Lucide, Heroicons 등)
- 폰트 (시스템 폰트 스택 vs 웹폰트)
- 컴포넌트 라이브러리 (shadcn/ui, Radix 등)
