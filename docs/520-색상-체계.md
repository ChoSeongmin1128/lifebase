# 색상 체계

## 목적
- Lifebase의 브랜드 색상, 시맨틱 색상, 다크/라이트 모드 색상을 정의한다.

## 브랜드 팔레트 (확정)

| 토큰 | 색상 | 용도 |
|------|------|------|
| `primary` | `#1b998b` | 브랜드 메인 (CTA 버튼, 활성 탭, 링크, 인터랙티브 요소) |
| `surface-accent` | `#f8f1ff` | 연한 라벤더 (포인트 배경, 호버, 선택된 날짜) |
| `surface-secondary` | `#decdf5` | 라벤더 (카드 구분, 카테고리 배경, 선택 상태) |
| `text-secondary` | `#656176` | 보라빛 회색 (보조 텍스트, 아이콘, 플레이스홀더) |
| `text-primary` | `#534d56` | 진한 보라 회색 (주 텍스트, 제목) |

## 적용 원칙 (프로토타입 2차)
- `primary`는 주요 액션, 활성 탭, 동기화 상태 핵심 강조에 우선 사용한다.
- `surface-accent`/`surface-secondary`는 보조 배경용으로 제한한다.
- 보조색은 완료 항목, 선택 날짜, 보조 정보 카드 같은 저우선 맥락 강조에만 사용한다.
- 경고/오류 상태는 보조색이 아니라 시맨틱 컬러(`warning`, `error`)를 우선 사용한다.

## 기본 색상 (브랜드 팔레트 외 자유 사용)

| 토큰 | 라이트 | 다크 | 용도 |
|------|--------|------|------|
| `background` | `#ffffff` | `#121212` | 메인 배경 |
| `surface` | `#ffffff` | `#1e1e1e` | 카드, 모달 배경 |
| `border` | `#e5e5e5` | `#2e2e2e` | 구분선, 보더 |
| `text-strong` | `#111111` | `#f5f5f5` | 강조 텍스트, 헤더 |
| `text-muted` | `#9ca3af` | `#6b7280` | 비활성 텍스트, 힌트 |

## 시맨틱 컬러

| 토큰 | 색상 | 용도 |
|------|------|------|
| `error` | `#ef4444` | 삭제, 에러, 높은 우선순위 |
| `warning` | `#f59e0b` | 경고, 중간 우선순위 |
| `success` | `#22c55e` | 성공, 완료, 동기화 완료 |
| `info` | `#3b82f6` | 안내, 링크(브랜드와 구분 필요 시) |
| `caution` | `#f97316` | 주의, 재인증 필요 |

- Google Calendar 캘린더 색상 24종과 호환
- colors.get() API로 런타임 조회, 하드코딩 금지
- Lifebase 자체 색상 체계와 Google 색상 간 매핑 테이블 필요

## 다크모드 / 라이트모드
- 기본값: 시스템 설정 따름
- 수동 전환 지원 (라이트/다크/시스템 3택)
- 구현: CSS 변수(Custom Properties) + `data-theme` 속성 전환
- 브랜드 팔레트 다크모드 조정:
  - `primary`: 동일 유지 (`#1b998b`)
  - `surface-accent`: 어둡게 조정 (예: `#2a2035`)
  - `surface-secondary`: 어둡게 조정 (예: `#352a40`)
  - `text-secondary`: 밝게 조정 (예: `#9e96a8`)
  - `text-primary`: 밝게 조정 (예: `#d4d0d8`)

## 열린 질문
- 다크모드 브랜드 색상의 정확한 값 (구현 시 시각 테스트 후 확정)
- Primary 호버/활성 상태 색상 변형 (밝게? 어둡게?)
