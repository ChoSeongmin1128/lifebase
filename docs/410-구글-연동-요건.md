# 구글 연동 요건

## 목적
- Google Calendar/Tasks 연동의 필수 구현 요건만 요약한다.

## 필수 요건
- OAuth 2.0 기반 사용자 승인
- Calendar 이벤트 CRUD 동기화
- Tasks 태스크 CRUD 동기화
- 동기화 실패 재시도(백오프)
- 충돌 발생 시 사용자 선택 가능 상태로 전환

## 캘린더 요건
- 반복 일정 처리(이번만/이후/전체)
- 색상(colorId) 보존
- 알림(useDefault 또는 overrides) 보존

## 태스크 요건
- 목록/태스크/하위태스크 구조 보존
- 완료 상태/완료 시각 동기화
- updated 기반 변경 감지

## 데이터 요건
- 외부 키(google_id), etag, updated 저장
- 마지막 동기화 시각/상태 저장
