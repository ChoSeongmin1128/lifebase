# 캘린더 기능

## 목적
- Lifebase 캘린더의 기능 범위를 정의한다.

## 상태
- 추후 상세 기획 예정

## 기능 범위 (초안)
- 일정 CRUD (제목, 시간, 장소, 메모)
- 연간/월간/주간/일간 뷰 전환
- 종일 일정 / 시간 지정 일정 구분
- 반복 일정 (매일/매주/매월/매년 + 커스텀 RRULE)
- 반복 일정 수정 (이번만/이후 전체/전체)
- 알림 설정 (다중 알림, 팝업/이메일)
- 다중 캘린더 (업무/개인 등 그룹 분리, 표시/숨김 토글)
- 캘린더별 색상, 이벤트별 색상 오버라이드
- Google Calendar 양방향 동기화
- 할 일 마감일 캘린더 표시 (연동)

## 열린 질문
- 드래그로 일정 이동/시간 변경 지원 범위 (웹만? 데스크톱도?)
- 빠른 일정 생성 UX (인라인 vs 모달)
- 미니 캘린더(사이드바 월간 미니맵) 포함 여부
- 참석자 기능 범위 (Google Calendar 참석자 표시만? 초대 기능도?)
