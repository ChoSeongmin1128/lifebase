# 마일스톤

## 목적
- 전체 기능을 구현하되, 의존성 순서대로 개발해 재작업을 줄인다.

## 입력
- 대상 플랫폼: Web, macOS Desktop, Windows Desktop, iOS, Android
- 대상 기능: 클라우드/공유/갤러리/동기화/캘린더/Todo/관리자
- 확정 스택: Go(chi, 헥사고날) + PostgreSQL + Redis(Asynq) + Next.js + Tauri v2 + React Native
- 운영 조건: Mac Mini(macOS, bare metal) + Thunderbolt 외장 12TB(APFS)
- 배포: bare metal + Caddy(HTTPS) + Cloudflare DNS 전용
- DB 마이그레이션: goose

### 최신 결정 (2026-02-27)
- Todo는 Google Tasks를 Source of Truth로 사용한다.
- 다중 Google 계정(개인/회사/가족) 동시 연결을 전제로 한다.
- 수동 정렬은 같은 계정 + 같은 Task List 내부에서만 허용한다.
- 계정 간 이동은 `copy + delete`로 처리한다.
- Todo 1차 범위는 Google Tasks 호환 필드(title/notes/due/status/parent)로 제한한다.

### 최신 업데이트 (2026-03-03)
- Calendar Month 뷰는 항상 42칸(6행) 고정 렌더링으로 통일하고, 해당 월이 아닌 날짜/일정도 약한 톤으로 노출한다.
- 28일 4주 완전 매칭 달은 `이전달 1줄 + 이번달 4줄 + 다음달 1줄` 규칙으로 시작 주를 1주 앞당긴다.
- Settings > Calendar에 `주 시작 요일`(`calendar_week_start`) 설정을 추가하고 Web/Mobile 캘린더 조회 범위/헤더에 동일 적용한다.
- Quick Create 팝오버를 날짜/시간 슬롯 클릭 지점 앵커 기준으로 열고, `datetime-local` 입력 폭 침범 이슈를 해소한다.
- Cloud/Gallery/Todo 상단 레이아웃을 `PageToolbar` 공통 컴포넌트로 정리해 화면 간 간격/선 위치 일관성을 맞춘다.
