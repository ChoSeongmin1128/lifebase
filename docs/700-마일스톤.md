# 마일스톤

## 목적
- 전체 기능을 구현하되, 의존성 순서대로 개발해 재작업을 줄인다.

## 입력
- 대상 플랫폼: Web, macOS Desktop, Windows Desktop, iOS, Android
- 대상 기능: 클라우드/공유/갤러리/동기화/캘린더/Todo/관리자
- 확정 스택: Go(chi, 헥사고날) + PostgreSQL + Redis(Asynq) + Next.js + Tauri v2 + React Native
- 운영 조건: Mac Mini(macOS, bare metal) + Thunderbolt 외장 12TB(APFS)
- 배포: bare metal + Caddy(HTTPS) + Cloudflare DNS 전용
- DB 마이그레이션: goose

## 결정

### 0단계 프로젝트 스캐폴딩
- pnpm workspace + Turborepo 모노레포 초기화
- Go 백엔드 skeleton (`cmd/server`, `internal/shared`, 헥사고날 디렉토리)
- PostgreSQL + Redis Homebrew 설정
- goose 마이그레이션 환경 구성
- `packages/domain`, `packages/api-types` 초기 구조
- Caddy + 도메인 + HTTPS 설정
- CI 파이프라인 (lint + test + build)

### 1단계 인증 + 사용자
- Google OAuth 2.0 흐름 구현 (서버)
- JWT 발급/갱신/만료 처리
- `users`, `user_accounts` 테이블 + 마이그레이션
- Next.js 앱 초기화 + 로그인 페이지
- 인증 미들웨어 (모든 API에 user_id 필터링)

### 2단계 클라우드 코어
- `files`, `folders`, `trash` 테이블
- 파일 업로드/다운로드/삭제/이동 API
- 폴더 CRUD + 계층 탐색
- 휴지통 (30일 자동 삭제)
- 사용자 할당량(1TB) 검증
- Web UI: Cloud 탭

### 3단계 갤러리 + 미디어 처리
- 미디어 타입 감지 + 썸네일 생성 워커 (Asynq)
- libvips 이미지 썸네일 + FFmpeg 비디오 썸네일
- Gallery API (필터/정렬)
- Web UI: Gallery 탭 (그리드/리스트/날짜별 뷰)

### 4단계 공유
- `shares`, `share_invites` 테이블
- 10분 초대 토큰 생성/검증
- ACL 기반 권한 (뷰어/에디터)
- Web UI: 공유 설정 모달, 초대 링크

### 5단계 캘린더
- `calendars`, `events`, `event_reminders`, `event_recurrences` 테이블
- 이벤트 CRUD API
- 반복 이벤트 확장 (RRULE)
- Google Calendar 동기화 워커 (webhook + syncToken)
- Web UI: 5개 뷰 (Year Compact/Timeline, Month, Week, Agenda)

### 6단계 Todo
- `todo_lists`, `todos` 테이블
- Todo CRUD + 계층(2단계) + 우선순위 + 고정
- 정렬/필터 API
- Google Tasks 폴링 동기화 워커
- Web UI: Todo 탭

### 7단계 Settings + 알림
- 사용자 설정 저장 API
- 알림 시스템 (in-app 토스트 + 알림 센터)
- Push 알림 (FCM Web)
- DND 스케줄 처리

### 8단계 Desktop 앱
- Tauri v2 프로젝트 초기화 (Web React 컴포넌트 재사용)
- 로컬 폴더 동기화 sidecar/워커
- 파일 변경 감지 + 업로드 큐
- 연결 끊김/복구 처리

### 9단계 Mobile 앱
- React Native (Expo bare) 초기화
- 하단 탭 네비게이션
- 사진 자동 업로드
- 캘린더/Todo 뷰 (모바일 최적화)

### 10단계 안정화
- E2E 테스트
- 에러 모니터링
- 성능 프로파일링 + 최적화
- 3플랫폼 QA
- 버전 `0.1.0` 태깅

## 열린 질문
