# 마일스톤

## 목적
- 전체 기능을 구현하되, 의존성 순서대로 개발해 재작업을 줄인다.

## 입력
- 대상 플랫폼: Web, macOS Desktop, Windows Desktop, iOS, Android
- 대상 기능: 홈/클라우드/공유/갤러리/동기화/캘린더/Todo/관리자
- 확정 스택: Go(chi, 헥사고날) + PostgreSQL + Redis(Asynq) + Next.js + Tauri v2 + React Native
- 운영 조건: Mac Mini(macOS, bare metal) + Thunderbolt 외장 12TB(APFS)
- 배포: bare metal + Caddy(HTTPS) + Cloudflare DNS 전용
- DB 마이그레이션: goose

### 최신 결정 (2026-02-27)
- Todo는 Google Tasks를 Source of Truth로 사용한다.
- 다중 Google 계정(개인/회사/가족) 동시 연결을 전제로 한다.
- 수동 정렬은 같은 계정 + 같은 Task List 내부에서만 허용한다.
- 계정 간 이동은 `copy + delete`로 처리한다.
- Todo 1차 범위는 Google Tasks 호환 필드(title/notes/due_date/status/parent)를 기준으로 하고, `due_time`과 `starred_at`은 LifeBase 로컬 확장으로 관리한다.

### 최신 업데이트 (2026-03-09)
- Todo due 모델은 `due_date` + 선택 `due_time`으로 재정의한다.
- Google Tasks 공개 API는 `due`를 RFC3339 문자열로 반환하지만 실제 의미는 date-only이므로, Google 왕복은 `due_date`만 사용한다.
- Todo parent/reorder는 Google `tasks.move`를 사용하고, 1단계를 넘는 원격 parent 체인은 최상위 부모 기준으로 정규화한다.
- Web Todo는 완료된 자식 항목도 활성 부모 아래에 유지해 Google Tasks와 동일한 부모 체인으로 표시한다.
- Web/Mobile Todo 리스트는 제목 아래 notes를 1~2줄 미리보기로 노출한다.
- Todo 정렬 기준은 `manual` / `due` / `recent_starred` / `title` 4종으로 통일하고, 기본값은 `due`로 둔다.
- Todo 삭제는 local push outbox 대기 중일 때 background pull sync가 같은 항목을 다시 살리지 않도록 보호한다.
- Web Todo 삭제는 우측 하단 Undo 토스트 5초를 거친 뒤 실제 삭제를 확정한다.
- Web Calendar 이벤트 삭제와 Web Cloud 파일/폴더 삭제도 같은 Undo 토스트 5초 패턴을 사용한다.
- Cloud 폴더 삭제/복원/휴지통 비우기는 하위 폴더·파일까지 재귀 처리하고, 휴지통도 폴더 내부 direct child 탐색을 지원한다.
- Cloud 원본 파일 실제 삭제 시 비어 있는 UUID prefix/user 디렉터리를 함께 정리하고, 비미디어 파일은 빈 썸네일 디렉터리를 만들지 않는다.
- Web Cloud 경로 헤더는 상단 breadcrumb 단일 구조로 정리하고, 폴더 이동 시 로컬 경로 캐시를 우선 사용해 헤더 깜박임을 줄인다.
- Web Cloud 잘못된/삭제된 폴더 링크는 빈 폴더로 보이지 않게 하고, invalid/not-found/error 상태를 구분해 렌더링한다.
- Web/Desktop Cloud는 `cmd/ctrl+a` 전체 선택과 다중 파일 `copy/cut/paste`, `Esc` 선택/클립보드 해제를 지원한다.
- 기본 작업 위치는 현재 repo의 `dev` 컨텍스트로 두고, worktree는 사용자 명시 요청 시에만 사용하는 규칙으로 조정한다.

- Todo: 리스트 칩 바, 완료 토글, 우선순위 표시, notes 미리보기, `due_date`/`due_time` 표시, 4개 정렬 기준
- Settings: 다크 모드/Push 알림/로그아웃
- Secure Store 기반 토큰 관리
- 미구현: native prebuild, 사진 자동 업로드, 모바일 최적화 뷰
