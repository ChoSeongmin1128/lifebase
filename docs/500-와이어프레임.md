# Design: Wireframes

## 목적
- 주요 플랫폼(Web/Desktop App/Mobile) 화면 뼈대를 빠르게 검증한다.

## 입력
- 캘린더/클라우드/Todo/Gallery/Settings 탭 공통 구조
- 캘린더 전역 계정 필터 + 완료 Todo 표시 토글
- 연간(Year) 뷰 2종 전환(Compact/Timeline)
- 연간 Timeline은 `calendar-year-view.html` 기준 밀도/레이아웃 정렬 필요
- 데스크톱 앱의 폴더 동기화(Nextcloud 유사) UX 필요
- 라이트/다크/시스템 테마 선택 토글
- 동기화 운영 지표는 기본 숨김, 문제 시 alert + 알림 목록 중심 노출
- Cloud는 대시보드보다 파일 브라우저 UX를 우선
- Gallery는 타입 필터/정렬 UX 포함
- Google OAuth 로그인 플로우 UI 별도 프로토타입 필요

## 결정
- Web 와이어프레임: `docs/prototypes/lifebase-web-view.html`
- Desktop App 와이어프레임: `docs/prototypes/lifebase-desktop-app-view.html`
- Mobile 와이어프레임: `docs/prototypes/lifebase-mobile-view.html`
- 캘린더 상세 Year View 실험: `docs/prototypes/calendar-year-view.html`
- 색상 비교/월간 샘플: `docs/prototypes/calendar-palette.html`
- Google 로그인 UI: `docs/prototypes/lifebase-google-login-view.html`
- Cloud 탭 기본 정보는 `동기화 상태/마지막 동기화/해결 필요 여부`로 축약
- `동기화 큐`, `충돌 파일 건수`는 고급 정보로 이동 (기본 화면 미노출)
- 충돌/오류는 alert와 알림 목록에 동시에 기록
- Cloud 메인 뷰는 파일 목록/폴더 탐색으로 고정하고, 특정 폴더 동기화 설정은 Settings > Cloud에서 관리
- 캘린더 Year는 `Compact`/`Timeline` 두 모드를 Settings > 캘린더 > 연간 보기 모드에서 전환
- Year Compact는 텍스트 없이 마커 + `+E/+T` 중심으로 구성
- Year Timeline은 좌측 날짜 전용 열 없이 셀 내부 날짜 라벨 중심으로 구성
- 캘린더 화면에는 핵심 탐색만 노출: 뷰 전환(Year/Month/Week) + 이전/오늘/다음 + 기간 라벨
- 설정성 옵션(계정 필터, 색상 범례, 완료 Todo 토글, Compact/Timeline 모드)은 모두 Settings 탭으로 이동
- Web/Desktop 사이드바는 접기/펼치기 토글 지원 (접히면 아이콘만 표시, `localStorage`에 상태 저장)
- 사이드바 아이콘은 인라인 SVG (Feather 스타일) 사용, 유니코드 이모지 사용 금지
- Settings 탭은 카테고리별 카드로 구성: 일반(계정/테마) / 캘린더(계정 필터/연간 보기 모드/색상 범례) / Todo(완료 표시) / Cloud(폴더 동기화)
- Web 레이아웃은 뷰포트 높이를 꽉 채우는 flex 레이아웃 사용 (하단 여백 제거)
- 주간(Week) 뷰는 08:00~22:00 1시간 단위 시간 그리드 + 종일 이벤트 영역 + 시간 블록 배치 방식 사용
- 모바일 Year Timeline은 내부 스크롤만 사용 (외부 `.content` 스크롤 차단으로 이중 스크롤 방지)

## 결정 (기술 세부)
- 모바일 Year Timeline의 `table-layout: fixed` 필수 (열 너비 균등 고정)
- Year Timeline의 `border-collapse: separate` + `border-spacing: 0` 사용 (sticky 헤더 호환)
- Year Timeline의 `th`는 `position: sticky; top: 0; z-index: 10` 이상으로 월 헤더 고정
- Year Timeline의 `td`는 `overflow: hidden; max-height: var(--year-row-height)` 로 행 높이 강제 고정
- `syncYearTimelineRowHeight()`는 `wrapper.clientHeight === 0`일 때 계산 스킵 (hidden 상태 보호)
- 탭 전환 시 캘린더 탭 진입하면 `syncYearTimelineRowHeight` 재실행
