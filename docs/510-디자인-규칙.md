# 디자인 규칙

## 목적
- LifeBase 전반의 디자인 방향과 원칙을 정의한다.

## 상태
- 프로토타입 기준 4차 규칙 반영 (2026-02-24)
- 프로토타입 5차: 캘린더 드롭다운 SVG+한글 워딩, 연간 빈 칸 투명화, Week 30분 spacer 규칙 반영 (2026-02-25)

## 디자인 방향 (초안)
- 깔끔하고 차분한 톤 (플랫 + 미니멀)
- 정보 밀도 적절히 유지 (클라우드/캘린더에서 과도한 여백 지양)
- 레이아웃: 사이드바 네비게이션 + 메인 콘텐츠 영역
- 탭 구조: Cloud / Calendar / Todo / Gallery / Settings 공통 탭 유지
- 데스크톱/웹 사이드바는 접기/펼치기 토글 지원 (펼침 208px / 접힘 52px 아이콘만)
- 모바일: 하단 탭바 중심
- 캘린더: 이벤트(바/텍스트)와 Todo(칩) 시각 분리 유지
- 캘린더 Year Compact: 텍스트 미노출, 마커+집계 중심 스캔 UI 유지
- 캘린더 Year Timeline: 정보 확인 UI로 유지, 중복 정보(좌측 별도 날짜열)는 제거
- 설정성 옵션(계정 필터, 색상 범례, 완료 Todo 토글)은 Settings 탭에서 통합 관리
- Compact/Timeline 모드는 캘린더 화면 드롭다운에서 직접 전환
- 운영 지표(예: 동기화 큐/충돌 건수)는 기본 숨김, 문제 발생 시 alert + 알림 목록으로 우선 노출
- 동기화 정상 상태에서는 정보량을 줄이고, 해결 필요 상태에서만 강한 시각 강조 사용
- 웹 레이아웃은 뷰포트 높이를 꽉 채우는 flex 기반 레이아웃 사용 (하단 여백 제거)
- Cloud 파일 테이블 헤더 셀 간 세로 구분선 표시 (컬럼 리사이즈 핸들 인지성 확보)
- Gallery는 앨범(기본)/리스트/상세 3가지 뷰 전환 지원, 앨범은 날짜별 그룹핑 썸네일 격자
- Cloud 드래그 이동 피드백: 폴더 드롭 타깃 hover 시 `surface-accent` 배경 + `primary` 테두리로 강조
- Cloud 드래그 이동은 `내 파일` 섹션에서만 노출하고, 최근/공유됨/중요/휴지통에서는 비활성 유지
- Cloud 컨텍스트 메뉴 우선순위: `복사/잘라내기/붙여넣기`를 상단에 배치하고 `이름 변경/이동/다운로드/삭제`를 하단에 배치
- Cloud 단축키 처리 규칙: mac(`cmd+x/c/v`), windows(`ctrl+x/c/v`) 지원, 입력 컴포넌트 포커스 시 단축키 가로채기 금지
- Cloud 복사 정책: 파일만 복사 허용, 폴더 복사 미지원
- In-App 알림 정책: 브라우저 기본 `alert/confirm/prompt` 금지, 토스트/배너/모달 패턴만 사용

## 사이드바
- Web/Desktop: 접기/펼치기 토글 지원
- 접힌 상태: 52px, 아이콘만 표시 (라벨 `opacity: 0`, `width: 0`)
- 펼친 상태: 208px, 아이콘 + 라벨
- 상태 저장: `localStorage('lifebase-sidebar')` (collapsed/expanded)
- 토글 버튼: 사이드바 상단 우측, 패널 레이아웃 아이콘

## 아이콘
- 모든 아이콘/상태 심볼/네비게이션 심볼은 SVG만 사용한다. (인라인 SVG 또는 SVG 스프라이트)
- 유니코드 이모지 사용 금지
- 유니코드 기호를 아이콘 대체로 사용하는 방식도 금지한다. (예: `↻`, `✓`, `⚠`, `▶`, `◂`, `▸`)
- 활성 탭: 아이콘 stroke #fff 전환
- 아이콘 목록: Cloud(구름), Calendar(캘린더+핀), Todo(체크마크 원), Gallery(이미지 프레임), Settings(톱니바퀴)

## Settings 구조
- 카테고리별 카드 레이아웃:
  - **일반**: Google 계정 연결, 테마 모드 (라이트/다크/시스템)
  - **캘린더**: 계정 필터, 주간 뷰 시간 범위 (시작 0~23시, 종료 1~24시), 색상 범례
  - **Todo**: 완료 Todo 표시 (숨김/표시), 기본 정렬 기준 선택
  - **알림**: Push 알림 켜기/끄기, DND 스케줄 (시작/종료 시간), "긴급만 허용" 토글
  - **Cloud**: 동기화 폴더별 카드(모드 pill + 경로 + 상태 + 액션) — 데스크톱 전체 기능, 모바일 읽기 전용
- 설정 변경은 즉시 반영 (저장 버튼 불필요)
- 동기화 모드 pill: 양방향(`--primary`), 업로드 전용(`--info`), 수동(`--text-muted`), 오류(`--error`)
- 오류 카드: `border-color: var(--error)` 강조, 충돌 해결/재시도/해제 액션 노출

## 타이포그래피
- 기본 폰트: Pretendard Variable (CDN dynamic subset)
- `font-family: "Pretendard Variable", Pretendard, -apple-system, BlinkMacSystemFont, sans-serif`

## 캘린더 화면 툴바
- 노출 항목: 단일 드롭다운 + chevron 이전/다음 아이콘 + 오늘 pill 버튼 + 기간 라벨
- 드롭다운 메뉴 (플랫 리스트): 연간 · 달력 / 연간 · 타임라인 / 구분선 / 월간 / 주간 / 일정
- 각 메뉴 항목에 SVG 아이콘(`.cvm-icon`) + 한글 라벨
- 드롭다운 버튼: `.cal-view-btn-icon`(뷰별 동적 SVG) + `.cal-view-btn-label`(한글 라벨)
- 활성 항목은 좌측 체크마크(SVG)로 표시
- 이전/다음은 chevron SVG 아이콘으로 표시
- 오늘 버튼: pill 형태(`border-radius: 999px`), 세 플랫폼 공통 `.cal-today-btn` 클래스 사용
- 제거 항목: 계정 필터, 완료 Todo 토글, 색상 범례 → Settings로 이동

## 캘린더 연간 Compact 빈 칸
- 해당 월에 속하지 않는 빈 칸(`.mini-day.empty`)은 배경/테두리 투명 처리
- 라이트/다크/시스템 테마 모두 동일 적용

## 캘린더 주간 뷰 시간 그리드
- 시작 시간 30분 전에 반 높이 spacer 행 추가 (dashed 약한 구분선, 라벨 없음)
- 시간 라벨은 정시(08, 09...)에만 셀 상단에 표시
- 이벤트 블록에 시작-종료 시간 범위 표시 (예: 09:00-10:00)
- 겹치는 이벤트는 같은 시간대에서 열 분할 나란히 배치

## 정렬 UI 패턴
- Cloud: 아이콘 드롭다운(툴바) + 테이블 헤더 클릭 (헤더와 드롭다운 양방향 동기화)
- Gallery: `<select>` 드롭다운 2개 (기준 + 방향)
- Todo: 정렬 아이콘 드롭다운(툴바), 수동 정렬 시 드래그 핸들 노출
- Calendar: 사용자 정렬 옵션 없음 (시스템 고정 규칙)
- 모바일 Cloud/Todo: 정렬 아이콘 → 하단 액션 시트 방식
- 모바일 Gallery: 네이티브 `<select>` 피커 유지
- 방향 표시: 활성 컬럼에 SVG 화살표 아이콘 (오름=위쪽, 내림=아래쪽)
- 터치 영역: 최소 44x44px

## Settings 추가 항목

- **Todo**: 완료 Todo 표시 (기존) + 기본 정렬 기준 선택
